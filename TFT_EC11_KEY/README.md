# TFT_EC11_KEY - EC11旋转编码器菜单系统

高内聚低耦合的EC11编码器+按键+U8G2显示菜单系统 for STM32F407

## 特性

- ✅ 模块化设计 (BSP/Middleware/App三层架构)
- ✅ 支持EC11旋转编码器(左旋/右旋/按键)
- ✅ 支持独立按键(短按/长按检测)
- ✅ 完整菜单系统(多级菜单/数值调节/开关切换)
- ✅ 支持U8G2显示库
- ✅ 提供标准库和HAL库两个版本
- ✅ 易于移植到其他平台

## 硬件连接

### EC11编码器
- A相 -> PA0
- B相 -> PA1
- 按键 -> PA2

### 显示器(U8G2)
- 根据具体显示器配置
